Foundry Virtual Tabletop - Version 9 Stable (9.249) Release Notes

This update provides an incremental patch for our recent Version 9 Stable release.

Bug Fixes

The Game Canvas

  • The autumn leaves weather effect once again appears on the entire scene rather than only in the top-left corner. (6510)
  • An edge-case issue which caused compression artifacts for the fog of war texture been resolved. (6622)

Interface and Applications

  • The tabs for configuration windows (such as the Drawing Configuration window) should once again render correctly. (6615)

Dice and Cards

  • Creating a new card in a deck no longer throws an error. (6616)
