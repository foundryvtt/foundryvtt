Game Entities¶

The term Entities is used to describe the individual records of data which comprise the Scenes, Actors, Items, Audio, and more which are featured within the World. This chapter introduces you to the different Entity types which are used within Foundry Virtual Tabletop and the API layers which allow you to modify them.
