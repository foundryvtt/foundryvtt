When Is This Necessary?

If a user enters an incorrect password on the World Login screen they'll see the following error message:

Invalid password provided for {username}!

And they'll be prevented from logging into the world. Thankfully there is an easy fix!

How to Reset User Passwords in Foundry VTT

If you're just getting started with Foundry VTT there is no password set on the default Gamemaster account that is made when you create a world or any other accounts that you create so try logging in without one to start. If you did set a password on your or another user's account you can reset them easily by following these steps:

  1. In V10: On the Setup screen find the world you would like to reset user passwords for and press Edit World.
    In V11: On the Setup screen find the world you would like to reset user passwords for and right click on it, then select Edit World
  2. Check the Reset User Passwords option.
  3. Press Update World

The next time you launch your world the passwords for all users will be reset and you and your users will be able to log in without a password.

Be sure to set a new password for your Gamemaster account and any other user accounts that you'd like to be secured as soon as possible to prevent anyone from making undesired changes to your Foundry VTT world.
