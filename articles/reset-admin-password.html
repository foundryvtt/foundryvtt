You may be prompted to enter a server Administrator password in two different screens:

  • The Foundry Login Screen
  • The World Login Screen

If you enter an invalid Administrator Password in either of these two screens, the same error message displays:

The provided server administrator password is invalid. If you have forgotten your configured password you will need to manually delete the Config/admin.txt file from your user data location.

First, Is There an Administrator Password Set?

When you are just getting started with Foundry VTT, for security reasons you may see this error message even though you have not set an Administrator password yet. The easiest way to double-check whether there currently is an Administrator password is to close and re-open Foundry.

  • If you are prompted for an Administrator password immediately when you launch Foundry, there is an Administrator password set.

    If you do not know this password, you will need to reset the Administrator password now. See Resetting Your Administrator Password below for instructions.


  • If you are not prompted for an Administrator password immediately when Foundry launches and the Setup screen displays instead, there is no Administrator password.

    Setting an Administrator password now is highly recommended. See Setting an Administrator Password below for instructions.

Resetting Your Administrator Password

Fortunately, if you can't remember your Administrator password, you can reset it yourself by finding and deleting a single file.

One of two methods may be used to locate this file:

The "Browse User Data" Method

  1. If Foundry isn't already running, launch it now
  2. Right-click the small Foundry icon in your Windows taskbar or Dock (typically located along the bottom of your screen)
  3. Select the "Browse User Data" option, then close Foundry
  4. In the folder that displayed, open the 'Config' folder and delete the 'admin' (or 'admin.txt') file inside it

Now, when you re-open Foundry again you should be able to access the 'Setup' screen and set a new Administrator password.

Finding Your Foundry User Data Folder Manually

This 'Config' folder that contains 'admin.txt' is located in your Foundry User Data folder. If you cannot open this folder using the "Browse User Data" method, please follow these instructions to find and delete the 'admin.txt' file instead.

If you use one of our official hosting partners instead of running Foundry directly on your own computer, this process may be different. If you use such a service and these instructions do not match what you are seeing, please contact your hosting service directly for assistance.

Setting an Administrator Password

In the Setup screen, you can set the Administrator password by clicking the gear-shaped "Configure" button located near the top-right of the screen.

Additional Information

On the World Login screen, when you attempt to use the "Return to Setup" button, you can encounter an "incorrect server Administrator password" error even though no Administrator password is set.

This might seem counterintuitive, but this is for security reasons. Throwing the generic "incorrect Administrator password" error prevents unauthorized access and also doesn't reveal that there is no server Administrator password. If you would like to use this 'Return to Setup' feature in the World Login screen, simply set an Administrator password.

Additionally, in the World Login screen, there is another way to get back to Setup without restarting Foundry or entering the Administrator password:

  1. Log in to the World as a Gamemaster-level account (if you don't know the password, try logging in without one)
  2. In the 'Settings' sidebar (the gear icon all the way at the end), click the 'Return to Setup' button at the bottom of the tab.
